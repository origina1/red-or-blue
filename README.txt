Для отработки навыков по HTML, CSS, JavaScript написал простую игру наподобие "Крестики-Нолики". Вместо крестиков красные клетки, вместо ноликов - синие.
Проект состоит из 2 HTML страниц:
1. index.html - страница приветствия с кнопкой "Начать игру". Код кнопки был найден в сети, изменил только цветовую гамму. Элемент TextScrumble также был позаимствован из открытых источников, так как понравился эффект скремблирования текста (не работает в IE).
2. game.html - страница самой игры "Красный или Синий". Код страницы и скрипта game.js полностью написал самостоятельно. При попытках сделать игровое поле квадратным пробовал пару вариантов, но оставил вариант с использованием библиотеки JQuery. Попытался сделать страницу с адаптивной разметкой (с помощью Flexbox), чтобы игровое поле подстраивалось под разные размеры окон браузера.
